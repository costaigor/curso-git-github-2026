git init . 			-> Cria um repo git na pasta atual
git status 			-> Mostra o status do trabalho 
git add nome_arquivo 		-> Adiciona arquivo para ser commitado
git commit -m 			-> Faz commit do que estava pra ser commitado
git log 			-> Mostra todo o histórico de commits

##FLUXO DE TRABALHO

2. iniciar o git dentro do diretório (git init .)
